–Я–Њ—З—В–Њ–≤—Л–є –Ї–ї–Є–µ–љ—В –≤ —В–µ—А–Љ–Є–љ–∞–ї–µ - Neomutt

–Э–∞—З–Є–љ–∞—П —А—Г–±—А–Є–Ї—Г –Њ–њ—В–Є–Љ–Є–Ј–Є—Ж–Є–Є —А–∞–±–Њ—З–Є—Е –њ—А–Њ—Ж–µ—Б—Б–Њ–≤, —Е–Њ—В–µ–ї–Њ—Б—М –±—Л —А–∞—Б—Б–Ї–∞–Ј–∞—В—М –Њ Neomutt - —Д–Њ—А–Ї —В–µ–Ї—Б—В–Њ–≤–Њ–≥–Њ –њ–Њ—З—В–Њ–≤–Њ–≥–Њ –Ї–ї–Є–µ–љ—В–∞ Mutt. –Ю—Б–Њ–±–Њ –≤–і–∞–≤–∞—В—Б—П –≤ –њ–Њ–і—А–Њ–±–љ–Њ—Б—В–Є Mutt —П –љ–µ –±—Г–і—Г, —Б–Ї–∞–ґ—Г –ї–Є—И—М —В–Њ–ї—М–Ї–Њ —З—В–Њ —Н—В–Њ–Љ—Г –њ–Њ—З—В–Њ–≤–Њ–Љ—Г –Ї–ї–Є–µ–љ—В—Г —Г–ґ–µ –±–Њ–ї—М—И–µ –і–≤—Г—Е –і–µ—Б—П—В–Є–ї–µ—В–Є–є –Є –і–Њ —Б–Є—Е –њ–Њ—А –Њ—Б—В–∞–µ—В—Б—П –њ—А–µ–і–њ–Њ—З—В–Є—В–µ–ї—М–љ—Л–Љ –њ–Њ—З—В–Њ–≤—Л–Љ –Ї–ї–Є–µ–љ—В–Њ–Љ –і–ї—П –±–Њ–ї—М—И–Њ–≥–Њ —З–Є—Б–ї–∞ –Њ–њ—Л—В–љ—Л—Е –њ–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї–µ–є. –Ф–∞–ї–µ–µ –Љ—Л –±—Г–і–µ–Љ –≥–Њ–≤–Њ—А–Є—В—М –Њ Neomutt(–і–∞–ї–µ–µ Nm).

–У–ї–∞–≤–љ–∞—П –Ј–∞–і–∞—З–∞ Nm –Ј–∞–Ї–ї—О—З–∞–µ—В—Б—П –≤ —В–Њ–Љ —З—В–Њ–±—Л –±—Л—В—М –њ–Њ—З—В–Њ–≤—Л–Љ –∞–≥–µ–љ—В–Њ–Љ –њ–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї—П. –§—Г–љ–Ї—Ж–Є–Њ–љ–∞–ї —Н—В–Њ–≥–Њ –Ї–ї–Є–µ–љ—В–∞ –њ—А–Є–Љ–µ—А–љ–Њ —В–∞–Ї–Њ–є:

  • –І—В–µ–љ–Є–µ –њ–Є—Б–µ–Љ, –Њ—В–њ—А–∞–≤–Ї–∞, –∞—А—Е–Є–≤–∞—Ж–Є—П –Є –і—А.
  • –Я–Њ–і–њ–Є—Б—Л–≤–∞–љ–Є–µ —Б–≤–Њ–Є—Е –њ–Є—Б–µ–Љ GPG-–Ї–ї—О—З–µ–Љ
  • –Я–Њ–і–і–µ—А–ґ–Ї–∞ –њ—А–Њ—В–Њ–Ї–Њ–ї–Њ–≤: POP3, IMAP - –і–ї—П –њ–Њ–ї—Г—З–µ–љ–Є—П –њ–Њ—З—В—Л –Є SMTP - –і–ї—П –µ—С –Њ—В–њ—А–∞–≤–Ї–Є
  • –Я–Њ–і–Ї–ї—О—З–µ–љ–Є–µ –љ–µ—Б–Ї–Њ–ї—М–Ї–Є—Е –∞–Ї–Ї–∞—Г–љ—В–Њ–≤
  • –С–µ–Ј–Њ–њ–∞—Б–љ–Њ—Б—В—М –∞–Ї–Ї–∞—Г–љ—В–∞ –њ–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї—П —Б –њ–Њ–Љ–Њ—Й—М—О —В–µ—Е–љ–Њ–ї–Њ–≥–Є–є PGP
  • –Я—А–Є–≤—П–Ј–Ї–Є –Ї–ї–∞–≤–Є—И, –Љ–∞–Ї—А–Њ—Б—Л –Ї–ї–∞–≤–Є–∞—В—Г—А—Л, –њ–Њ—З—В–Њ–≤—Л–µ –њ–Њ—В–Њ–Ї–Є –Є —В.–і.

–≠—В–Њ –і–∞–ї–µ–Ї–Њ –љ–µ –≤—Б–µ –≤–Њ–Ј–Љ–Њ–ґ–љ–Њ—Б—В–Є Nm –Є —З—В–Њ–±—Л —Г–Ј–љ–∞—В—М –Њ –љ–Є—Е –±–Њ–ї—М—И–µ –і–ї—П –љ–∞—З–∞–ї–∞ —Г—Б—В–∞–љ–Њ–≤–Є–Љ –µ–≥–Њ.

–£—Б—В–∞–љ–Њ–≤–Ї–∞

–ѓ –±—Г–і—Г –њ—А–Њ–Є–Ј–≤–Њ–і–Є—В—М —Г—Б—В–∞–љ–Њ–≤–Ї—Г –Є –љ–∞—Б—В—А–Њ–є–Ї—Г –≤ Arch. –£—Б—В–∞–љ–Њ–≤–Ї–∞ –љ–∞ —А–∞–Ј–љ—Л–µ —Б–Є—Б—В–µ–Љ—Л –Њ—В–ї–Є—З–∞–µ—В—Б—П, –љ–Њ –љ–∞—Б—В—А–Њ–є–Ї–∞ —Б–∞–Љ–Њ–≥–Њ Nm –Њ–і–Є–љ–∞–Ї–Њ–≤–∞ –і–ї—П –≤—Б–µ—Е.

Linux

Arch:

$ pacman -S neomutt

Debian:

$ apt-get install neomutt

Homebrew:

$ brew install neomutt

–Я—А–Њ—Ж–µ—Б—Б —Г—Б—В–∞–љ–Њ–≤–Ї–Є –і–ї—П –і—А—Г–≥–Є—Е —Б–Є—Б—В–µ–Љ –Љ–Њ–ґ–љ–Њ –њ–Њ—Б–Љ–Њ—В—А–µ—В—М –љ–∞ –Њ—Д–Є—Ж–Є–∞–ї—М–љ–Њ–Љ —Б–∞–є—В–µ NM

Windows

–£—Б—В–∞–љ–Њ–≤–Ї–∞ –і–Њ—Б—В—Г–њ–љ–∞ –≤ –≤–Є–і–µ –њ–∞–Ї–µ—В–∞ Cygwin. –і–ї—П –µ–≥–Њ —Г—Б—В–∞–љ–Њ–≤–Ї–Є –≤–Њ–Ј—М–Љ–Є—В–µ –њ–Њ—Б–ї–µ–і–љ—О—О –≤–µ—А—Б–Є—О —Г—Б—В–∞–љ–Њ–≤–Ї–Є Cygwin –Є –≤—Л–±–µ—А–Є—В–µ –њ–∞–Ї–µ—В NeoMutt.

MacOS

–І—В–Њ–±—Л —Г—Б—В–∞–љ–Њ–≤–Є—В—М neomutt –≤ MacOS —Г –≤–∞—Б –і–Њ–ї–ґ–µ–љ –±—Л—В—М —Г—Б—В–∞–љ–Њ–≤–ї–µ–љ MacPorts.

$ sudo port install neomutt

–Э–∞—З–∞–ї—М–љ–∞—П –љ–∞—Б—В—А–Њ–є–Ї–∞ –њ–Њ—З—В—Л

–Ф–ї—П –њ—А–Є–Љ–µ—А–∞ –і–Њ–±–∞–≤–Є–Љ –≤–Њ–Ј–Љ–Њ–ґ–љ–Њ—Б—В—М —З–Є—В–∞—В—М –њ–Є—Б—М–Љ–∞ —Б Gmail. –Ф–ї—П –љ–∞—З–∞–ї–∞ –љ–µ–Љ–љ–Њ–≥–Њ –љ–∞—Б—В—А–Њ–Є–Љ –љ–∞—И —П—Й–Є–Ї.

–Я–µ—А–µ–Љ–µ—Й–∞–µ–Љ—Б—П –≤ –љ–∞—И –њ–Њ—З—В–Њ–≤—Л–є —П—Й–Є–Ї, –љ–∞–ґ–Є–Љ–∞–µ–Љ –љ–∞ –љ–∞—Б—В—А–Њ–є–Ї–Є –≤ –Њ—В–Ї—А—Л–≤—И–µ–Љ—Б—П –Њ–Ї–љ–µ –љ–∞–ґ–Є–Љ–∞–µ–Љ –љ–∞ “–Т—Б–µ –љ–∞—Б—В—А–Њ–є–Ї–Є” –Ф–∞–ї–µ–µ –љ–∞–ґ–Є–Љ–∞–µ–Љ –љ–∞ –≤–Ї–ї–∞–і–Ї—Г “–Я–µ—А–µ—Б—Л–ї–Ї–∞ –Є POP/IMAP” > –Ф–Њ—Б—В—Г–њ –њ–Њ –њ—А–Њ—В–Њ–Ї–Њ–ї—Г IMAP: –Т–Ї–ї—О—З–Є—В—М IMAP > –°–Њ—Е—А–∞–љ—П–µ–Љ –Є–Ј–Љ–µ–љ–µ–љ–Є—П.

–Э–∞ —Н—В–Њ–Љ –љ–∞—Б—В—А–Њ–є–Ї–∞ —П—Й–Є–Ї–∞ –Ј–∞–Ї–Њ–љ—З–µ–љ–∞.

–Ъ–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П

–Я–µ—А–µ—Е–Њ–і–Є–Љ –Ї –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є Nm. –Ф–ї—П –љ–∞—З–∞–ї–∞ –љ–∞–Љ –љ—Г–ґ–љ–Њ —Б–Њ–Ј–і–∞—В—М —Д–∞–є–ї –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є

1$ cd ~
2$ mkdir .mutt
3$ cd .mutt/
4$ vim .neomuttrc

–Ґ–µ–њ–µ—А—М –Ј–∞–њ–Є—И–µ–Љ –њ–∞—А—Г –Њ—Б–љ–Њ–≤–љ—Л—Е –љ–∞—Б—В—А–Њ–µ–Ї –і–ї—П –њ–Њ–і–Ї–ї—О—З–µ–љ–Є—П IMAP –Є SMTP.

–ѓ –љ–µ –±—Г–і—Г –≤–і–∞–≤–∞—В—М—Б—П –≤ –њ–Њ–і—А–Њ–±–љ–Њ—Б—В–Є –љ–∞—Б—В—А–Њ–Ї–Є –і–ї—П –Ї–∞–ґ–і–Њ–є —Г—З–µ—В–љ–Њ–є –Ј–∞–њ–Є—Б–Є, –њЊ—В–Њ–Љ—Г —З—В–Њ –Ј–∞—Б—В–∞–≤–Є—В—М neomutt –і–µ–є—Б—В–≤–Є—В–µ–ї—М–љ–Њ –Њ–±—Й–∞—В—М—Б—П —Б –≤–∞—И–Є–Љ –њ–Њ—З—В–Њ–≤—Л–Љ —Б–µ—А–≤–µ—А–Њ–Љ вАФ —Н—В–Њ —Б–Њ–≤—Б–µ–Љ –і—А—Г–≥–Њ–µ –і–µ–ї–Њ, –і–ї—П –Ї–Њ—В–Њ—А–Њ–≥–Њ —Б—Г—Й–µ—Б—В–≤—Г—О—В –і–µ—Б—П—В–Ї–Є —А—Г–Ї–Њ–≤–Њ–і—Б—В–≤.

–Э–∞—Б—В—А–Њ–є–Ї–∞ IMAP:

1set imap_user="user@example.com"
2set imap_pass="pass"
3set folder=imaps://imap.gmail.com:993
4set spoolfile=+INBOX
5set editor = "vim"

–Т folder —Г–Ї–∞–Ј—Л–≤–∞–µ–Љ IMAP-—Б–µ—А–≤–µ—А –њ–Њ—З—В—Л, –∞ –≤ spoolfile вАФ –Ї–∞—В–∞–ї–Њ–≥ —Б –њ–Њ—З—В–Њ–є –љ–∞ —Н—В–Њ–Љ —Б–µ—А–≤–µ—А–µ. Editor —Г–Ї–∞–Ј—Л–≤–∞–µ—В –љ–∞ —В–Њ, –Ї–∞–Ї–Њ–є —А–µ–і–∞–Ї—В–Њ—А –±—Г–і–µ—В –Є—Б–њ–Њ–ї—М–Ј–Њ–≤–∞—В—М—Б—П –≤–љ—Г—В—А–Є Nm.

–І—В–Њ –±—Л –њ–µ—А–Є–Њ–і–Є—З–µ—Б–Ї–Є –њ—А–Њ–≤–µ—А—П—В—М –≤—Б–µ –Ї–∞—В–∞–ї–Њ–≥–Є вАФ —Г–Ї–∞–Ј—Л–≤–∞–µ–Љ:

set imap_check_subscribed

–Ф–ї—П –Ї–µ—И–Є—А–Њ–≤–∞–љ–Є—П –і–∞–љ–љ—Л—Е –ї–Њ–Ї–∞–ї—М–љ—Л—Е, —З—В–Њ –±—Л –љ–µ —В—П–љ—Г—В—М –і–∞–љ–љ—Л–µ –Ї–∞–ґ–і—Л–є —А–∞–Ј —Б —Б–µ—А–≤–µ—А–∞ –і–Њ–±–∞–≤–ї—П–µ–Љ:

set header_cache = ~/.cache/mutt
set message_cachedir = ~/.cache/mutt

–Э–∞—Б—В—А–Њ–є–Ї–∞ SMPT

–Ф–ї—П –Њ—В–њ—А–∞–≤–Ї–Є –њ–Њ—З—В—Л c –њ–Њ–і–і–µ—А–ґ–Ї–Њ–є SSL –Є–Ј Nm —Г–Ї–∞–Ј—Л–≤–∞–µ–Љ:

set smtp_pass = "pass"
set smtp_url=smtps://useraddress@smtp.gmail.com:465
set ssl_force_tls = yes

set realname = "name"
set from = "user@example.com"
set use_from = yes

–Э–∞—З–∞–ї—М–љ–∞—П –љ–∞—Б—В—А–Њ–є–Ї–∞ –Ј–∞–≤–µ—А—И–µ–љ–∞. –°–Њ—Е—А–∞–љ—П–µ–Љ –Є –Ј–∞–Ї—А—Л–≤–∞–µ–Љ —А–µ–і–∞–Ї—В–Њ—А –Є –Ј–∞—Е–Њ–і–Є–Љ –≤ Nm.

–Э–µ–±–Њ–ї—М—И–∞—П —И–њ–∞—А–≥–∞–ї–Ї–∞ –њ–Њ Nm

–У–ї–∞–≤–љ–Њ–µ –Љ–µ–љ—О:

  • q: –≤—Л—Е–Њ–і –Є–Ј –њ—А–Њ—Б–Љ–Њ—В—А–∞ –Є–ї –њ—А–Њ–≥—А–∞–Љ–Љ—Л
  • d: –Њ—В–Љ–µ—В–Є—В—М –љ–∞ —Г–і–∞–ї–µ–љ–Є–µ (–±—Г–і–µ—В —Г–і–∞–ї–µ–љ–Њ –њ—А–Є –≤—Л—Е–Њ–і–µ)
  • m: —Б–Њ–Ј–і–∞—В—М –љ–Њ–≤–Њ–µ –њ–Є—Б—М–Љ–Њ
  • r: –Њ—В–≤–µ—В–Є—В—М –љ–∞ —В–µ–Ї—Г—Й–µ–µ

–Я—А–Є –њ—А–Њ—Б–Љ–Њ—В—А–µ –њ–Є—Б—М–Љ–∞:

  • i: –≤–µ—А–љ—Г—В—М—Б—П –Ї —Б–њ–Є—Б–Ї—Г –њ–Є—Б–µ–Љ –≤ —П—Й–Є–Ї–µ
  • –њ—А–Њ–±–µ–ї: —Б–ї–µ–і—Г—О—Й–∞—П —Б—В—А–∞–љ–Є—Ж–∞ —В–µ–Ї—Б—В–∞
  • -: –њ—А–µ–і—Л–і—Й–∞—П —Б—В—А–∞–љ–Є—Ж–∞ –њ–Є—Б—М–Љ–∞
  • r: –Њ—В–≤–µ—В–Є—В—М
  • d: —Г–і–∞–ї–Є—В—М
  • —Б—В—А–µ–ї–Ї–Є –≤–≤–µ—А—Е/–≤–љ–Є–Ј/–≤–ї–µ–≤–Њ/–≤–њ—А–∞–≤–Њ: —Б–ї–µ–і—Г—О—Й–µ–µ/–њ—А–µ–і—Л–і—Г—Й–µ–µ –њ–Є—Б—М–Љ–Њ

–С–µ–Ј–Њ–њ–∞—Б–љ–Њ—Б—В—М

–Я—А–Є —Б–Њ–Ј–і–∞–љ–Є–Є –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –Љ—Л —Б–Њ–Ј–і–∞–ї–Є –Њ–≥—А–Њ–Љ–љ—Г—О –і—Л—А—Г –≤ –±–µ–Ј–Њ–њ–∞—Б–љ–Њ—Б—В–Є –љ–∞—И–µ–є –њ–Њ—З—В—Л. –Э–∞ –Љ–Њ–є –≤–Ј–≥–ї—П–і —Е—А–∞–љ–Є—В—М –њ–∞—А–Њ–ї—М –Њ—В –њ–Њ—З—В—Л –≤ –Њ—В–Ї—А—Л—В–Њ–Љ –≤–Є–і–µ - –Њ–њ–∞—Б–љ–Њ. –Ф–∞–≤–∞–є—В–µ —Н—В–Њ –Є—Б–њ—А–∞–≤–Є–Љ.

–Ф–ї—П –і–Њ—Б—В—Г–њ–∞ –Ї –њ–Њ—З—В–µ –≤–љ—Г—В—А–Є Nm —П –±—Г–і—Г –Є—Б–њ–Њ–ї—М–Ј–Њ–≤–∞—В—М GPG-–Ї–ї—О—З–Є. –°–Њ–Ј–і–∞–і–Є–Љ —Д–∞–є–ї –і–ї—П —Е—А–∞–љ–µ–љ–Є—П –њ–∞—А–Њ–ї—П

1$ cd .mutt/
2$ vim passwords

—Б–Њ —Б–ї–µ–і—Г—О—Й–Є–Љ —Б–Њ–і–µ—А–ґ–Є–Љ—Л–Љ

1set imap_pass="password"
2set smtp_pass="password"

–Я—А–Њ—Б—В–Њ –≤–њ–Є—Б—Л–≤–∞–µ–Љ —Б–≤–Њ–Є –њ–∞—А–Њ–ї–Є

–Ф–∞–ї–µ–µ —Б –њ–Њ–Љ–Њ—Й—М—О GPG –Ј–∞—И–Є—Д—А—Г–µ–Љ —Н—В–Њ—В —Д–∞–є–ї

–Х—Б–ї–Є —Г –≤–∞—Б –љ–µ—В GPG-–Ї–ї—О—З–µ–є - –њ—А–Є–і–µ—В—Б—П –Њ–±–Ј–∞–≤–µ—Б—В–Є—Б—М

1$ gpg --gen-key

–Я–Њ—Б–ї–µ —Б–Њ–Ј–і–∞–љ–Є–µ –Ї–ї—О—З–µ–є –њ–µ—А–µ—Е–Њ–і–Є–Љ –љ–µ–њ–Њ—Б—А–µ–і—Б—В–≤–µ–љ–љ–Њ –Ї —И–Є—Д—А–Њ–≤–∞–љ–Є—О

1$ gpg -r your.email@example.com -e ~/.mutt/passwords
2$ ls ~/.mutt/passwords*
3/home/user/.mutt/passwords   /home/user/.mutt/passwords.gpg
4$ shred ~/.mutt/passwords
5$ rm ~/.mutt/passwords

–Ґ–µ–њ–µ—А—М –Ј–∞–Љ–µ–љ–Є–Љ smtp_pass –Є imap_pass –≤ —Д–∞–є–ї–µ .neomuttrc –љ–∞

source "gpg -d ~/.mutt/passwords.gpg |"
