Tmux - терминальный мультиплексор

2022-04-24
0 minute read
cli , tmux , tools

tmux это терминальный мультиплексор, который позволяет работать с несколькими консольными сессиями в одном окне.

Настройка

Установим tmux в систему

sudo pacman -S tmux

Сразу создадим конфигурационный файл

cd .config
mkdir tmux
cd tmux
touch tmux.conf
nvim tmux.conf
